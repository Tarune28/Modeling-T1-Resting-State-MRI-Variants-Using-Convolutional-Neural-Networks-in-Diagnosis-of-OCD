Overview & Usage

The SZD model was compiled with an overall accuracy of 82.08%. Furthermore heatmaps generated by GradCam plotted greater importance on the right frontal lobe as compared to the remainder of the scan. Displacement of the heat spot off the resting-state MRI is a result of the global average pooling layer being utilized for weights. The file input below allows for the model to be trialed. Certain input sizes may be rejected by the model. (1 voxel @ 199 x 137 is the optimal input). Alternate disorders can be inputted in order to further understanding of the model.
